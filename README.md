# React Based Timer App

A simple React based application that allows user to Start, Stop and Reset the timer.
Build to practice Web Development Skills like React Hooks and Component UI based development.
Used CSS for minimalist User Interface.

## Features :
* Start, Stop and Reset features
* Real-time updates using `useState` and `useEffect`
* Clean and minimal user interface
* Responsive design

## Tech Stack/ Technologies Used :
* React : To create app and enhance User Interface.
* JavaScript (ES6) : Cleaner syntax and better structure to applications.
* CSS : Enhancing the design.

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/SamJais7/react-based-timer-app.git
```
### 2. Navigate to project folder
```bash
cd react-based-timer-app
```
### 3. Install Dependencies
```bash
npm install
```
### 4. Run the application
```bash
npm start
```
### 5. Open In browser
```bash
http://localhost:3000
```
